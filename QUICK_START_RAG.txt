═══════════════════════════════════════════════════════════════════
TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

Problem: "Cannot reach the AI backend"
→ Make sure Terminal 2 shows "✅ API Server is ready"
→ Restart Python: Ctrl+C in Terminal 2, then "python rag_api.py"

Problem: Port 8080 already in use
→ Find: netstat -ano | findstr :8080
→ Or edit rag_api.py to use port 8081

Problem: Ollama not responding
→ Check Terminal 1 is running "ollama serve"
→ If stuck, close and rerun

Problem: Response is very slow (>10 seconds)
→ RTX 5060 Ti is slow (8GB VRAM limit)
→ First inference takes longer (model warmup)
→ Subsequent queries are faster (cached)
