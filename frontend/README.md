# COMPGame - Computer Science Learning Games

A gamified platform for learning computer science concepts through interactive games.

## Setup Instructions for Windows

### Prerequisites

- Node.js (v16 or newer)
- Git
- Visual Studio Code (recommended)

### Installation

1. **Clone or extract the repository**

2. **Run the setup script**
   
  Double-click on `setup.bat` or run it from Command Prompt:

    2.1 Open the project in VSCode
    2.2 Open a terminal (Ctrl + `)
    2.3 Run the batch file by typing:
        `.\setup.bat`

  After the setup is complete, you can start the development server with `npm run dev` as instructed by the script.
